NexusHR - Admin Dashboard Template

Thank you for purchasing NexusHR. This documentation will guide you through customization.

1. Folder Structure

The template is divided into assets/css, assets/js, and assets/images.

2. Changing Colors

Open assets/css/style.css. At the top, you will find the :root variables. Change --primary to your brand color.

3. Data Integration

The template uses a Mock Data system in assets/js/main.js. Look for the loadMockData() function to connect your API.
